## **Exercices de compréhension**

### **Exercice 1 : Prédire la sortie**

Pour chaque boucle `for`, prédisez la sortie **sans utiliser Python**. Ensuite, vérifiez vos réponses en exécutant le
code.

1.

```python
for i in range(5):
    print(i)
```

2.

```python
for fruit in ["pomme", "banane", "cerise"]:
    print(fruit)
```

3.

```python
for i in range(2, 6):
    print(i * 2)
```

4.

```python
for i in range(0, 10, 3):
    print(i)
```
